Guess the Number - RANKED

by Clyton

Rules

  • Rule 1: Guess the number within the chosen range.
  • Rule 2: The game tracks the number of attempts.
  • Rule 3: The leaderboard ranks based on difficulty and tries.
  • Rule 4: Avoid using profanity in the names and stay professional.
  • Rule 5: Use the feedback section to give reports or feedbacks.
  • Note : The score is tracked on number of guesses, time taken, range difficulty (higher difficulty = more rewarding.)
